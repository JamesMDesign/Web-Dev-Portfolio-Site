Events Organiser App

Made using Html, Bootstrap, Mongo DB

This was a team project for myself and one other over the course of a week and my introduction to databases and backend development for web.
The user can register with the site, log in and then create, modify and delete events with classfields they can set for the title, event description as well as functionality to attach a picture for each event.
