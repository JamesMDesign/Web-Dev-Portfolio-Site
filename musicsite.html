Music Streaming Site

Made using Html, Bootstrap, React, Mongo DB

This was our final project set for the Developer Academy full stack development course. We worked in teams of four over a month to create a site of our choosing. We decided to make a music streaming site with music recommendations, playlists, user profile and tags for recommendations using the Spotify API.

This was a significant task for us as 4 inexperienced and newly trained software developers and took all the skills we had learned to date to pull off. I was really pleased with the result and as a team we overcame lots of hurdles and challenges, completing our main goals of creating a functional site using the Spotify API including the core functionality we planned on implementing from the initial planning sessions.
